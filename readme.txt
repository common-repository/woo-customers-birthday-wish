== Description ==
This is an excellent plugin for Woocommerce that offers an option to add the date of birth at the time of registration process. While the customer can sign up for a new account on your system it shows an additional option to enter their Date of Birth with other mandatory details of the registration form.

It has wonderful modules that are easy to manage the admin section of the website. In Addition to this, it automatically generates a birthday email for the registered users to wish them a joyful birthday.

How does it work?

The Plugin Continuously matches the current date with the date of births of registered users in the database. And if the today's Date match with the DOB of any user then it will generate an automated email to the user to give them birthday wishes from the website owner.

Features:

* Send Automated Emails to the Customer on their birthday

* Provide Functionality to  Enter DOB on the Registration Time.

* Customers can edit their Date of birth anytime by visiting edit page of their profile.

* Shown a calendar menu to select the date of birth in form of date, month & year.




== Installation ==

Installing "Birthday Message For Woocommerce" can be done either by searching for "Birthday Message For Woocommerce" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
